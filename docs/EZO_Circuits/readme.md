## EZO EC (Electrical Conductivity)

### Data type (EC,TDS,SAL,SG)
- EC	Electrical Conductivity (μS - microSiemens)
- TDS 	Total Dissolved solids (PPM)
- SAL	Salinity (PSU - practical salinity uni)
- SG	Specific Gravity
